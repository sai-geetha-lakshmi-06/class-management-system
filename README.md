# class-management-system
The Class Management System helps store and manage student details like name, roll number,and marks. It allows adding, viewing,searching,updating,and deleting records. This project reduces manual work, improves accuracy, and organizes class information using simple c programming concepts.
Class Management System

Abstract

The Class Management System is a simple C program designed to store, manage, and organize student details such as name, roll number, and marks. It allows users to add, view, search, update, and delete records, making classroom data handling easier, faster, and more accurate.


---

Features of the Program

Add new student records

Display all student details

Search for a student using roll number

Update existing student information

Delete a student record

Simple text-based interface

Error handling for invalid inputs



---

Functional Requirements

The system must store student details in structured format.

Users should be able to add, view, modify, or delete records.

Input validation must be performed for roll numbers and options.

Search functionality must retrieve correct student data.



---

Technical Requirements

Language: C

Compiler: GCC / MinGW / Turbo C / Online C compiler

Header Files: stdio.h, string.h, stdlib.h

Works on Windows, Linux, or macOS



---

How to Run the Code

1. Download or clone the repository


2. Open the .c file in any C IDE or compiler


3. Compile the program:

gcc class_management.c -o cms


4. Run the program:

./cms


5. Use the menu to perform class management operation
